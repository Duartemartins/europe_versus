# Brand Concept 2: Swiss Brutalist Data

## Design Philosophy
Ultra-clean, high-contrast Swiss design meets brutalist web aesthetics. Data is presented with ruthless clarity—no decoration, pure information architecture. Inspired by Helvetica documentaries, Swiss railway timetables, and modern data art.

## Visual Direction

### Color Palette
- **Primary**: Pure black (#000000) — maximum contrast
- **Secondary**: Pure white (#FFFFFF) — clean backgrounds
- **Accent**: Electric blue (#0066FF) — single accent for Europe
- **Warning**: Signal yellow (#FFCC00) — for attention points
- **Subtle**: Light gray (#F5F5F5) — backgrounds for sections

### Typography
- **Headlines**: "Helvetica Neue" or "Inter" (tight tracking, bold weight)
- **Body**: "Helvetica Neue" light — Swiss precision
- **Data**: "Space Mono" — monospaced for perfect alignment
- **Labels**: "Helvetica Neue" condensed, uppercase

### Layout Principles
- **Strict grid**: 12-column, mathematical precision
- **No curves**: Square corners, hard edges
- **Dense information**: Efficient use of space
- **Strong hierarchy**: Size and weight, not color, create hierarchy
- **Systematic**: Everything aligned, nothing arbitrary

### Key Design Elements
1. **Header**: Black bar, white logo, minimal text
2. **Cards**: White or light gray, black borders (2px solid)
3. **Charts**: Black/white with single blue accent for Europe
4. **Numbers**: Oversized, monospaced, left-aligned
5. **Rules**: Thick black horizontal dividers between sections

### Tone & Voice
- "Just the facts"
- Clinical precision
- Confident through restraint
- Anti-decoration

### Differentiation
What makes this memorable: **Looks like it was designed by a Swiss engineer, not an AI.** The extreme simplicity and systematic approach creates trust through clarity.

## Sample UI Elements

### Hero Section
```
EUROPE
VERSUS

```

### Data Cards
- 2px black border
- White background
- Oversized metric name (uppercase)
- Giant number for Europe value
- Smaller numbers for comparisons, aligned right
- No shadows, no gradients

### Navigation
- Black bar, white text
- Underline on hover (no color change)
- All caps, letter-spaced
